    STDIO Wrapper¶

    mcpgateway.wrapper is a lightweight MCP-compatible stdio server shipped inside the main package (mcp-contextforge-gateway). It mirrors the tools, prompts and resources that live in an MCP Gateway catalog and re-publishes them via stdin/stdout so that any MCP client â€” even those without SSE support or JWT headers â€” can call them locally (e.g. Claude Desktop, Cline, Continue).

    • Transparent tool proxy â€“ wrapper â†’ Gateway HTTP RPC â†’ tool; results stream back to stdout.
